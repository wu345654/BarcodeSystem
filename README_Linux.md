# 条码系统 Linux 打包说明

## 打包步骤

### 1. 在Linux系统上执行打包脚本

```bash
# 进入项目目录
cd /path/to/BarcodeSystem

# 运行打包脚本
python3 build_linux.py
```

### 2. 打包完成后的文件结构

打包完成后，会在 `dist` 目录中生成以下文件：

- `BarcodeSystem` - 可执行文件
- `order_system.db` - 数据库文件
- `run_barcode_system.sh` - 一键运行脚本

### 3. 运行系统

```bash
# 进入dist目录
cd dist

# 运行一键脚本
./run_barcode_system.sh
```

系统将在 `http://localhost:5001` 上运行。

## 注意事项

1. **系统要求**：
   - Linux系统
   - Python 3.7+
   - 网络连接（用于安装依赖）

2. **依赖项**：
   - PyInstaller
   - 项目中 `requirements.txt` 列出的所有依赖

3. **首次运行**：
   - 系统会自动创建数据库文件（如果不存在）
   - 首次访问系统时，会自动初始化数据库表结构

4. **端口设置**：
   - 默认端口为 5001
   - 如需修改端口，请编辑 `app.py` 文件中的端口配置

5. **防火墙**：
   - 确保防火墙允许 5001 端口的访问

## 故障排除

- **打包失败**：检查依赖项是否正确安装
- **运行失败**：检查数据库文件是否存在，端口是否被占用
- **访问失败**：检查防火墙设置，确保 5001 端口已开放
